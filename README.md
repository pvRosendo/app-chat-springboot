Este projeto está sendo desenvolvido para ser uma aplicação de chat em tempo real com Websocket utilizando Java e Spring . A aplicação deve oferecer funcionalidades básicas similares às encontradas em plataformas de chat como o Discord. O foco principal é a criação de um backend robusto e escalável, utilizando as abstrações e boas práticas.
